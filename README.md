# AI-Assisted PCOS Risk Prediction System

## Project Overview
This project aims to develop an AI-assisted biomarker-based system for early PCOS risk assessment and prediction using computational approaches.

## Objective
To analyze clinical biomarkers and patient parameters to identify patterns associated with Polycystic Ovary Syndrome (PCOS).

## Potential Parameters
- Age
- BMI
- Menstrual cycle length
- Hormonal markers
- Insulin-related parameters
- Clinical symptoms

## Tools Planned
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Streamlit
- GitHub

## Status
Project initiated – literature review and dataset collection phase.
